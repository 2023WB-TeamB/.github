<br/>

#### 1. clone the repository(docker-GCP)<br>
```
git clone https://github.com/2023WB-TeamB/Docker-GCP.git
```

<br/>

#### 2. Set environment file<br>
Path : Docker-GCP/backend/.env<br>
    
```

GPT_SECRET_KEY=''

ENG_TITLE_ASSISTANT_ID=''
```
